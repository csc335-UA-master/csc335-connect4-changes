# Connect4 Changes

Read the PDF on D2L for the project details.

To bring your code from project 1 into this project:
1. Right click the project name in the Package Explorer and pick "Import..". 
2. Select "General->File System" from the treeview. 
3. Click "Next". 
4. Click "Browse" to navigate to your project 1 folder (probably up one level and inside the git directory in a subdirectory called "csc335-connect4". 
5. Select the src folder. 
6. In the resulting dialog, click to check all of the files and finish the import.
